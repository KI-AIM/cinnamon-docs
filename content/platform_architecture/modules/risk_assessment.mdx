# Cinnamon Risk-Assessment

The risk assessment module provides methods for the privacy evaluation of datasets.

## Tasks

Cinnamon Risk-Assessment is responsible for the following tasks:

### Privacy Evaluation

Cinnamon Risk-Assessment provides two types of assessments, the risk assessment  and the general assessment.

## Architecture

This module is based on Python and Fast Api 0.115.
Processes are stored in memory only so no database is used.

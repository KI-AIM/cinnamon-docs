# Cinnamon Evalution

The evaluation module analyses single datasets and compares tow datasets, i.e. the original and the anonymized dataset, by calculating statistics and metrics.

## Tasks

Cinnamon Evaluation is responsible for the following tasks:

### Calculation of general statistics

### Evaluation of machine learning utility

### Evaluation of resemblance

## Architecture

Cinnamon Evalution uses Python and Flask 3.
Processes are stored in memory only so no database is used.
